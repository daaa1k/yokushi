# Plan: yokushi - AI Agent Hook Tool

## Context

Build a Rust binary `yokushi` (抑止 = suppress/deter) that acts as a `PreToolUse` hook
for AI coding agents (Claude Code, OpenCode, etc.). When an agent attempts to use a tool,
the hook validates against user-configured suppression rules and blocks matches with a
clear reason. Output behavior is configurable per agent type.

Reference pattern: https://zenn.dev/kawarimidoll/articles/7da5fd40f19fb1

## Hook Protocol

**Input** (stdin JSON from any agent):
```json
{
  "session_id": "...",
  "hook_event_name": "PreToolUse",
  "tool_name": "Bash",
  "tool_input": { "command": "git push origin main" }
}
```

**Block - Claude Code** (stdout JSON, exit 0):
```json
{
  "hookSpecificOutput": {
    "hookEventName": "PreToolUse",
    "permissionDecision": "deny",
    "permissionDecisionReason": "Command suppressed (pattern: 'git push'): ..."
  }
}
```

**Block - default** (stderr, exit 2):
```
yokushi: blocked by rule 'git push': Direct git push is prohibited.
```

**Allow**: no output, exit 0.

## Config Format (`yokushi.yaml`)

```yaml
version: "1"

# Agent-specific output behavior
agents:
  claude-code:
    output: json          # JSON to stdout + exit 0 (Claude Code official format)
  default:
    output: exit          # message to stderr + exit 2 (works with most agents)

# Suppression rules
rules:
  # Bash tool (default when tool is omitted)
  - pattern: "git push"
    message: "Direct git push is prohibited. Please use pull requests."

  - pattern: "\\bawk\\b"
    message: "Use of 'awk' is prohibited. Use 'rg' instead."

  # Other tool types
  - tool: "Write"
    # field defaults to "file_path" for Write/Edit/Read tools
    pattern: "\\.env$"
    message: "Writing to .env files is prohibited."

  - tool: "WebFetch"
    # field defaults to "url" for WebFetch
    pattern: "example\\.com"
    message: "Access to example.com is restricted."
```

**Config discovery order** (first found wins):
1. `--config <path>` CLI flag
2. `YOKUSHI_CONFIG` environment variable
3. `./yokushi.yaml` (current directory)
4. `~/.config/yokushi/config.yaml`

If no config found → allow all (silent exit 0)

## Data Structures

### `config.rs`

```rust
pub struct Rule {
    pub tool: Option<String>,    // default: "Bash"
    pub field: Option<String>,   // default: auto-detected per tool
```

```rust
pub enum OutputMode { Json, Exit }

pub trait OutputStrategy {
    fn block(&self, event: &str, reason: &str) -> !;
    fn allow(&self) -> !;
}

// DefaultOutput: stderr + exit 2
```

## Field Defaults Per Tool

```rust
fn default_field(tool: &str) -> &str {
    match tool {
        "Bash" => "command",
        "Write" | "Edit" | "Read" | "Glob" | "Grep" => "file_path",
        "WebFetch" => "url",
        "WebSearch" => "query",
        "Task" => "prompt",
        _ => "command",
    }
}
```
